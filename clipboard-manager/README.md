# 📋 Copypad — コピペリスト管理ツール

コピペしたいテキストを **ファイル → グループ（属性）→ 項目** の3階層で管理し、**ワンタッチでコピー** できるツールサイトです。
**ドラッグで並び替え**、そして **チームでのリアルタイム同期（任意）** に対応しています。

**公開URL:** https://1411998888z-ai.github.io/no.1/

### 階層イメージ

```
ファイル（例: よく使う定型文）
├─ グループ「メール」
│   ├─ 項目: お礼メール 冒頭
│   └─ 項目: 自分のメールアドレス
└─ グループ「署名」
    └─ 項目: 会社署名
```

## 特長

- 🗂️ **ファイル → グループ → 項目の3階層** — ファイルの中を「属性1」「属性2」のようにグループ分けして整理
- 📋 **ワンタッチコピー** — カードのボタンを押すだけでクリップボードにコピー
- 🔀 **ドラッグで並び替え** — ハンドル `⠿` でファイル・グループ・項目を並び替え。**項目はグループをまたいで移動**も可能（PC・スマホ対応）
- ☁️ **チーム同期（任意）** — Firebase 設定を入れると、URLを知っている全員で同じリストをリアルタイム共有・共同編集
- 🔍 **検索** — ラベル・本文をまたいで絞り込み（`Ctrl/⌘ + K` でフォーカス）
- 💾 **自動保存** — 入力内容は常にブラウザにも保存（オフラインでも閲覧可）
- ⬇️⬆️ **エクスポート / インポート** — 全データを JSON で書き出し・読み込み
- 🌓 **ライト / ダーク** — OS設定に自動追従＋手動切替

## 使い方

1. 公開URL（または `index.html`）をブラウザで開きます。
2. 左の「＋」で **ファイル** を追加。
3. 「＋ グループを追加」で **グループ（属性）** を作成。
4. 各グループの「＋ 項目」で **コピペ項目** を登録（追加時にグループも選べます）。
5. カードの「📋 コピー」でクリップボードにコピー。
6. ハンドル `⠿` をドラッグして並び替え／項目をグループ間で移動。

### ショートカット

| キー | 動作 |
| --- | --- |
| `Ctrl / ⌘ + K` | 検索にフォーカス |
| `Ctrl / ⌘ + Enter` | 編集モーダルで保存 |
| `Esc` | モーダル / サイドバーを閉じる |

## ☁️ チーム同期を有効にする（任意・無料）

同期を設定しない場合は、これまで通り **各ブラウザ内（localStorage）だけ** で動作します。
同期を有効にすると、**同じワークスペースを開いている全員が、同じリストをリアルタイムで共有・共同編集** できます。

### 手順

1. [Firebase コンソール](https://console.firebase.google.com/) でプロジェクトを作成
2. **Firestore Database** を作成（ロケーションは任意）
3. プロジェクト設定 → 「アプリを追加」→ **ウェブ (`</>`)** を選び、表示される `firebaseConfig` の値をコピー
4. `js/firebase-config.js` の `window.COPYPAD_FIREBASE_CONFIG` に値を貼り付けて保存（コミット）
5. Firestore の **セキュリティルール** を以下に設定して公開：

   ```
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /workspaces/{doc} {
         allow read, write: if true;
       }
     }
   }
   ```

設定後にサイトを開くと、右上のステータスが **「● 同期中」** になります。

### 共有範囲について

- `js/firebase-config.js` の `COPYPAD_WORKSPACE_ID`（既定 `team-main`）が「共有の単位」です。
  同じIDを見ている人どうしが同じデータを共有します。別チーム用に分けたいときはIDを変えてください。
- **注意（プライバシー）:** 上記ルールはログイン不要の「オープン共有」です。サイトURL＋設定は公開情報のため、
  URLを知っていれば誰でも読み書きできます。限定したい場合は Firebase Authentication を使う／ワークスペースIDを
  推測されにくい文字列にする、などの対策を検討してください（ご希望あれば実装します）。
- 同時編集は「後から保存した内容が優先（ドキュメント単位の上書き）」です。小規模チームでの利用を想定しています。

### セキュリティ上の補足

`firebase-config.js` に入る `apiKey` などは**公開前提のクライアント識別子**であり、秘密鍵ではありません。
リポジトリに含めても問題ありません（アクセス制御は上記の Firestore ルールで行います）。

## 構成

```
clipboard-manager/
├─ index.html            # 画面
├─ css/style.css         # スタイル（トークン / レスポンシブ / ダークモード）
├─ js/
│  ├─ app.js             # ロジック（ファイル/グループ/項目・保存・コピー・並び替え・同期）
│  ├─ firebase-config.js # Firebase 設定（空ならローカルのみ）
│  └─ vendor/
│     └─ Sortable.min.js # ドラッグ並び替えライブラリ（同梱）
└─ README.md
```

ドラッグ並び替えは [SortableJS](https://github.com/SortableJS/Sortable)（MIT、同梱）、
同期は [Firebase Firestore](https://firebase.google.com/)（CDN読み込み・任意）を使用しています。
