# もち太 シーン静止画

ここに ChatGPT等で作った「もち太の職業シーン静止画」を置きます。
リール生成時、これらを Kling で動画化します。

## 置き方

- ファイル名を **ナレーション順** にする：`1.png`, `2.png`, `3.png` ...
- 投稿文を「。」で区切った文の数だけ用意するのが理想
  - 例：5文の投稿なら `1.png`〜`5.png`
  - 画像が足りない場合は最後の画像を使い回します
- 縦長(9:16)に近い構図が望ましい（リールは1080x1920）
- もち太がそのシーンを演じている絵（例：占い師なら水晶玉の前のもち太）

## 推奨

各文の内容に合ったシーンを1枚ずつ。もち太の見た目はChatGPT側で固定されるので、
同じもち太が各シーンで動く動画になります。
