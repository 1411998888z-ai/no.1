# X投稿候補 自動生成bot（→ LINEに毎朝配信）

@_x_saku_ 用の投稿アシスタント。毎朝7時(JST)に、インプが取れそうな投稿案を3つLINEに届けます。
あなたは届いた案を精査してXに投稿するだけ。

## 構成

- **GitHub Actions**（毎朝cron実行・無料枠で十分）
- **Google Gemini API**（投稿生成・無料枠で十分）
- **LINE Messaging API**（配信・月200通無料、毎日1通なら余裕）

すべて完全無料。

## セットアップ

[SETUP.md](./SETUP.md) を参照してください（15分）。

## 投稿の型

アカウントのトーンに合わせて、以下5つの型からランダムに3つを選び生成します：

1. 逆張り断定型（「〜するな。〜しろ。」）
2. 共感→気づき型
3. 箇条書きノウハウ型
4. 体験談→学び型
5. 脳科学/心理学解説型

型を調整したい場合は `generate_posts.py` の `POST_PATTERNS` を編集してください。

## ファイル

- `generate_posts.py` — 投稿生成 & LINE配信スクリプト
- `.github/workflows/daily.yml` — 毎朝の自動実行
- `SETUP.md` — 初期セットアップ手順
