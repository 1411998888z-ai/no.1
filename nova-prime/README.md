# Nova Prime｜STEP 0 カリキュラム設計シート

契約者向けの面談前アンケートページ。設計レポート（全8 STEP・21問／V字感情アーク）を実装したもの。

- `index.html` … 単体で動く完結型ページ（依存ライブラリなし）。ブラウザで開くだけで動作します。

## 実装で押さえた設計意図

| レポートの狙い | 実装 |
| --- | --- |
| I 進行感の演出 | 8 STEPページ送り＋上部の進捗バー（%・STEP x/8） |
| プライミング（記入例） | 各設問に「記入例」を常時表示。回答の長さ・方向を誘導 |
| 空白期間の不安解消 | 入力を都度 `localStorage` に自動保存。中断→再開で消えない |
| G 不安の先出し | STEP 0-6 のセクション名は「サポートを最適化するために」。Q17 に「同じ轍を踏まない設計をします」を明示 |
| J 社会的証明 | Q13 に「どちらのタイプでも成果を出している先輩がいます」を明示 |
| V字の右端 | 完了画面で STEP 1（初回面談）へ視線を送る。Q21 の回答をそのまま提示 |

**回答者に見せないもの**（＝レポート内の講師向けメモ）は実装に含めていません。
MECHタグ／「→ 選択者は〜」の適性判定メモ／設計コメントはすべて非表示です。
